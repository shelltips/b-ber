# b-ber-reader


## Install

Clone the repository and install dependencies

```
$ yarn
```

## Develop

Watch for changes in the `src` directory and start a server at `http://localhost:3000/`

```
$ yarn start
```

## Build

Compile assets to `dist`

```
$ yarn build
```

## Test

Run tests with [`jest`](https://facebook.github.io/jest/)

```
$ yarn test
```
